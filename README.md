# Commercial Insights for a Pharmaceutical Marketplace

## Overview
This project simulates commercial analytics for a healthcare marketplace serving
emerging markets. It focuses on buyer behavior, supplier pricing and marketplace
performance using synthetic pharmaceutical transaction data.

## Key Questions
- Who are the most valuable buyers?
- How does supplier pricing affect volume?
- What indicators reflect marketplace health?

## Tools
- Python (pandas, numpy)
- SQLite (SQL-style analysis)
- Google Colab : Open here - [![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/fjnelima-pixel/commercial-insights-pharma-marketplace/blob/main/commercial_insights_pharma_marketplace.ipynb)

- GitHub

## Key Insights
- Buyer spend is concentrated among a small subset of accounts.
- Moderate pricing correlates with higher supplier volume.
- Demand patterns vary across therapeutic classes.
